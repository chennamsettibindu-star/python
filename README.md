## Hangman Game
A simple text-based Hangman game where the player guesses a hidden word one letter at a time. The game provides six attempts to guess the correct word.
Concepts Used:
- Random Module
- While Loop
- If-Else Statements
- Strings
- Lists
## Stock Portfolio Tracker
A Python program that calculates the total investment value of a stock portfolio using predefined stock prices. The program also saves the portfolio summary to a text file.
Concepts Used:
- Dictionary
- User Input
- Arithmetic Operations
- File Handling
## Task Automation with Python Scripts
A simple automation script that extracts email addresses from a text file and saves them into another text file using Regular Expressions.
Concepts Used:
- Regular Expressions (re)
- File Handling
- Input/Output
 ## Basic Chatbot
A rule-based chatbot that responds to basic user inputs such as greetings and exits when the user types "bye".
Concepts Used:
- If-Else Statements
- Loops
- User Input/Output
  ##  Technologies Used
